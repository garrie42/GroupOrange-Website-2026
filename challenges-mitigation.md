# Challenges & Mitigation

## HIPAA / compliance expectations
- Strong access control
- Audit logging
- Secure transmission and storage

## Adoption & integration
- Make the workflow simple for staff
- Support integration via APIs where possible
- Provide value even when full integration is not available

## Reliability
- Offline access with syncing to support weak connectivity
